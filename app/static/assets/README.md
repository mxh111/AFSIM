# Web Assets

This directory is reserved for browser-ready map and symbol assets derived from legally available local AFSIM resources.

Expected future asset types:

- Lightweight GeoJSON coastlines, roads, borders, hydrography, and terrain contours.
- Web-friendly raster/vector tiles exported from local AFSIM map resources.
- glTF or sprite assets converted from approved AFSIM model/icon sources.
- Symbol atlases for MIL-STD-like tactical rendering.

Do not place or overwrite files inside the AFSIM installation directory. Keep source notes and conversion steps with any asset added here.
